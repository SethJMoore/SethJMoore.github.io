--- layout: post title: Let GitHub host it. date: '2014-04-10T19:49:00.001-05:00' author: Seth Moore tags: modified_time: '2014-04-10T19:49:33.093-05:00' blogger_id: tag:blogger.com,1999:blog-24777982.post-1883971241739922473 blogger_orig_url: http://blog.munjaros.com/2014/04/let-github-host-it.html --- I've been looking into creating my own personal landing page and pointing my sethjmoore.com domain there, rather than at my Flavors.me page. This would allow me to brush up on my web development skills and work toward having a more useful page without having to start paying Flavors.me for more than their basic functionality. It seemed that I would have to start paying for hosting somewhere else, but then I found out about GitHub Pages, where I could host my website as a git repository and serve from my domain for free. So today I created a basic landing page and set up my DNS so that www.sethjmoore.com goes to it. It's a start.
